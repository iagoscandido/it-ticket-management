# Personal It Tickets Management

## Requirements

- Node.js 24.14.0 LTS
- Docker Desktop

## How to run

```sh
     # starts postgres
     docker compose up
     # installs dependencies
     npm i
     # starts a local server on http://localhost:8080
     # docs on http://localhost:8080/docs
     npm run dev
     # stats drizzle studio on https://local.drizzle.studio
     npm run db:studio
```
